# 🤖 AI-Driven Chatbot Platform with Full DevOps Pipeline

## 📌 Project Overview

This project aims to design, develop, and deploy a scalable AI-driven chatbot platform using modern DevOps practices. The chatbot will be integrated with AWS Lex for conversational AI, containerized using Docker, deployed on Amazon EKS using Kubernetes, and automated with Jenkins CI/CD pipelines.

---

## 🎯 Project Objectives

- Build an AI-powered chatbot backend using Python Flask.
- Develop a responsive frontend using React.
- Containerize the application using Docker.
- Automate infrastructure provisioning using Terraform.
- Deploy the application on Amazon EKS.
- Implement CI/CD using Jenkins.
- Integrate AWS Lex for chatbot functionality.
- Monitor the application using Amazon CloudWatch.
